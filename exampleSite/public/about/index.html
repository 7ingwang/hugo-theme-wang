我是王珽

UX / UI 设计师、Web 前端工程师

从2013年开始从事设计专业，我和Design的关系始于我十几岁的时候，我开始将我儿时的绘画技能和我对科技的好奇心结合起来，自学图形工具和网络技术。出于这个原因，我一直把我的设计研究集中在计算设计上。

在我7年的工作经验中，我面对过不同的工作环境:机构、初创公司、内部设计团队。在这些不同的环境中工作让我学会了在做出任何设计决定之前深入了解项目的背景。

环境是指导项目设计决策的唯一规则。我的方法包括在设计数字产品时，在审美愉悦、产品价值和商业成果之间找到适当的平衡。这种方法可以通过理解设计思维(科学)、设计制作(工艺)和设计业务(设计思维)来实现。
